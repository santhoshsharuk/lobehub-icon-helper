Installation

VS Code Marketplace

Install directly from the VS Code Extensions view.

ext install santhoshsharuk.lobehub-icon-helper Install

Manual Installation

Download and install the .vsix file manually.

  1. Download the latest release
  2. Open VS Code & Run "Install from VSIX..."
  3. Select the downloaded file

About

LobeHub Icon Helper was created to streamline the developer workflow when working with LobeHub's comprehensive icon library. Built with performance and usability in mind, this extension eliminates the friction between finding the perfect icon and implementing it in your project. Developed by Santhosh Sharuk, this extension represents a commitment to improving developer productivity in the VS Code ecosystem.
